= Create New Schema

== Краткое описание
Актер начинает работу с чистого листа: открывает меню `Schema`, выбирает `New` и получает заготовку схемы, с которой можно сразу продолжать предметную работу. Вместо пустого технического холста редактор сразу выдает минимальный стартовый набор объектов, чтобы было от чего отталкиваться при обсуждении системы.

== Текущее состояние
Сценарий уже реализован и начинается с подтверждения сброса текущей работы. После этого редактор открывает новую схему в центре экрана и переводит актера в состояние, где можно сразу менять название схемы, двигать converter и перестраивать flows.

== Что подтверждено
- `createEmptySchema()` создает стартовую схему с metadata, одним converter, двумя внешними flows и одним value element.
- После команды `New` редактор выбирает схему как текущий объект, центрирует вид и заново показывает рабочую поверхность.
- Заголовок новой схемы по умолчанию - `New Value System`.

== Замечания аналитика
В исходниках подтвержден только стандартный browser confirmation dialog. Отдельный сценарий на сохранение перед сбросом или на восстановление несохраненной работы не найден.

== Требуемые изменения
Новых изменений не требуется.
