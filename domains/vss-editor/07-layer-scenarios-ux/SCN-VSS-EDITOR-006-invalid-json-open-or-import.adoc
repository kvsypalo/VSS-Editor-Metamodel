= Invalid JSON Open Or Import

== Краткое описание
Актер пытается открыть или добавить файл, который оказался невалидным JSON. В этом случае редактор завершает действие явным отказом, сообщает об ошибке и сохраняет уже открытую схему в рабочем состоянии.

== Текущее состояние
Отказной маршрут уже реализован и одинаково работает для `Open` и `Add Schema`. Если файл не читается как JSON, редактор сохраняет текущую схему как основное рабочее состояние и завершает операцию сообщением об ошибке.

== Что подтверждено
- Отказной маршрут одинаково доступен из `Open` и `Add Schema`.
- При parse error редактор показывает явный сигнал `Invalid JSON`.
- Текущая рабочая схема остается на месте, потому что изменение state выполняется только после успешного чтения файла.

== Замечания аналитика
Source basis подтверждает базовый отказной маршрут через alert-сообщение и сохранение текущей сессии.

== Требуемые изменения
Новых изменений не требуется.
