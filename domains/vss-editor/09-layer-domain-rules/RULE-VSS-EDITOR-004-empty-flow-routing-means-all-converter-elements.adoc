= Empty Flow Routing Means All Converter Elements

== Краткое описание
Если у flow нет явного списка `element_type_ids`, editor трактует это как прохождение всех element types соответствующего converter. При первом открытии flow properties panel пустой список автоматически заполняется всеми доступными elements converter, а снятие всех checkbox возвращает flow к значению "все типы по умолчанию".

== Текущее состояние
Правило реализовано в `renderFlowProps(flow)`. Ветка инициализации проверяет, пуст ли `element_type_ids`, и заполняет его `allTypes.map(e => e.id)`, а helper text объясняет пользователю, что пустой список означает использование всех типов по умолчанию.

== Что подтверждено
- Flow routing UI строится на списке elements того converter, который связан с flow.
- Пустой или неинициализированный `element_type_ids` один раз заполняется всеми IDs converter elements.
- Hint прямо сообщает: `If none selected, all are used by default.`

== Замечания аналитика
Правило выводится одновременно из UI hints и состояния данных, поэтому materialized как самостоятельное domain rule.

== Требуемые изменения
Новых изменений не требуется.
