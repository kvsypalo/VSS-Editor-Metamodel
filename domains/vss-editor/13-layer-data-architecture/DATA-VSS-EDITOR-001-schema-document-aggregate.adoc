= Schema Document Aggregate

== Краткое описание
Schema document - корневой переносимый объект редактора. Он объединяет metadata схемы и коллекции `nodes`, `flows`, `elements` и `texts`, которые вместе описывают полную diagram state текущей value-system schema.

== Текущее состояние
В текущей реализации schema aggregate живет в `state.schema`, сериализуется целиком в JSON и заменяется при `Open`. Default aggregate создается функцией `createEmptySchema()`.

== Что подтверждено
- Корневой объект содержит раздел `schema` с `id`, `title`, `description`, `version`, `author`, `created_at`, `updated_at`.
- Aggregate содержит массивы `nodes`, `flows`, `elements` и `texts`.
- Export JSON сохраняет именно этот aggregate целиком.

== Замечания аналитика
Source basis подтверждает JSON structure и lifecycle aggregate, но не показывает отдельную версионируемую schema contract documentation вне кода.

== Требуемые изменения
Новых изменений не требуется.
