= Text Annotation

== Краткое описание
Text annotation - свободный текстовый объект на canvas, который не меняет graph semantics flows и converters, но позволяет actor добавлять комментарии и контекст к схеме. Он имеет собственные координаты и редактируемое текстовое содержимое.

== Текущее состояние
Text object создается через кнопку `Add Text`, хранится в массиве `texts`, рендерится как SVG text element и может перемещаться drag-and-drop независимо от converters.

== Что подтверждено
- Text object хранит `id`, `x`, `y`, `text`.
- Add Text создает новый объект в центре текущего canvas view.
- Text object имеет собственную properties panel с редактированием текста и удалением.

== Замечания аналитика
Объект намеренно отделен от schema metadata и node descriptions, потому что исходный продукт дает ему отдельный lifecycle и отдельное взаимодействие на canvas.

== Требуемые изменения
Новых изменений не требуется.
