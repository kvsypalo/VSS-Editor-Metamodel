= Edit Diagram Objects And Properties

== Краткое описание
Редактор должен поддерживать прямое изменение структуры схемы и свойств выбранных объектов. Actor должен иметь возможность добавлять converters, внешние потоки, text annotations, менять свойства schema, node, flow и text, а также управлять выбором и перемещением объектов на canvas.

== Текущее состояние
Canvas и properties panel покрывают редактирование schema metadata, node properties, flow routing и text objects. Режимы `Hand` и `Arrow`, drag, rectangle selection, zoom и pan обеспечивают работу с диаграммой как с интерактивной поверхностью.

== Что подтверждено
- Node card позволяет менять имя, описание, BPMN link, external schema link и добавлять следующие или внешние flows.
- Flow card позволяет менять тип, value, unit, color, source/target nodes и routing element types.
- Add Text создает свободный text object, который можно перемещать и редактировать.

== Замечания аналитика
Requirement описывает текущую behavior surface одной страницы и не делит редактор на отдельные feature modules.

== Требуемые изменения
Новых изменений не требуется.
