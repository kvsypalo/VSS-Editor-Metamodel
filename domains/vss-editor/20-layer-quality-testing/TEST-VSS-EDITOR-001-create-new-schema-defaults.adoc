= Create New Schema Defaults Test

== Краткое описание
Тест проверяет, что команда `New` создает default schema aggregate и переводит редактор в предсказуемое начальное состояние. Проверка закрывает primary scenario создания новой схемы.

== Текущее состояние
Тест пока materialized как validating test specification по подтвержденному коду. Автоматический test harness в source basis не обнаружен.

== Что подтверждено
- `createEmptySchema()` определяет ожидаемый состав default объектов.
- `btnNew.onclick` выполняет полный reset editor state после подтверждения пользователя.
- После reset вызываются `selectSchema`, `centerView` и `render`.

== Замечания аналитика
Тест описывает подтвержденную проверку на уровне продукта и кода в текущей wave.

== Требуемые изменения
