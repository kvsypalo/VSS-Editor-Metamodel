= Edit Schema Objects Test

== Краткое описание
Тест проверяет основной editing route редактора: изменение свойств schema, node, flow и text, а также манипуляции canvas selection/drag/zoom. Проверка закрывает primary scenario редактирования схемы на canvas.

== Текущее состояние
Тест materialized как validating specification по подтвержденным forms и interaction handlers в `app.js`.

== Что подтверждено
- Properties panel имеет отдельные rendering ветки для schema, node, flow и text.
- Editing controls меняют соответствующие поля объектов и вызывают `updateTimestamp`, `renderDiagram` или `render`.
- Canvas handlers покрывают drag nodes, drag text, pan, rectangle selection и zoom.

== Замечания аналитика
Тест описывает интегральную проверку editor behavior без дробления на micro-tests каждого контрола.

== Требуемые изменения
Новых изменений не требуется.
