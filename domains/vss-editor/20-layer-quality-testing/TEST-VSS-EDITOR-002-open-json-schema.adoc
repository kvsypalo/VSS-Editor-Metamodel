= Open JSON Schema Test

== Краткое описание
Тест проверяет, что команда `Open` загружает корректный JSON-файл схемы и делает его новым текущим editor state. Проверка закрывает primary scenario открытия существующей схемы.

== Текущее состояние
Тест materialized как validating test specification по подтвержденному поведению `fileMode = "open"` и ветке успешного parse JSON.

== Что подтверждено
- JSON-файл читается через `FileReader` и `JSON.parse`.
- Ветка `open` заменяет `state.schema` загруженным объектом.
- После загрузки вызываются `selectSchema`, `centerView` и `render`.

== Замечания аналитика
Тест не покрывает failure parse; эта ветка вынесена в отдельный failure scenario и отдельный test object.

== Требуемые изменения
Новых изменений не требуется.
