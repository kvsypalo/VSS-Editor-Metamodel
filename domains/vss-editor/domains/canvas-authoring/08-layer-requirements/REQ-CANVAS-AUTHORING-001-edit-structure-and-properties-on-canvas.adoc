= Edit Structure And Properties On Canvas

== Краткое описание
Редактор должен поддерживать прямое уточнение структуры схемы и свойств объектов прямо на canvas. Актор должен иметь возможность управлять converters, flows, element types и text annotations в одном интерактивном authoring loop без выхода из рабочей схемы.

== Текущее состояние
В текущей реализации этот contract уже выполняется через связку canvas handlers и properties panel. Изменения к schema, node, flow и text объектам сразу применяются к открытой схеме и видны в той же рабочей поверхности.

== Что подтверждено
- Canvas interaction и properties panel работают как единый editing surface.
- Правки структуры и свойств происходят над одной и той же открытой схемой.
- Graph rules локального authoring route подтверждены отдельными parent-level `RULE-*` карточками.

== Замечания аналитика
Requirement child domain удерживает локальный authoring contract в целом. Детализированные object-level acceptance branches уже раскрыты в parent domain data/rule/test cards.

== Требуемые изменения
Новых изменений не требуется.
