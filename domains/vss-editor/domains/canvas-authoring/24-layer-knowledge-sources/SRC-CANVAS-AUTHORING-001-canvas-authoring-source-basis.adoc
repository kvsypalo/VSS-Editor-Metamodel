= Canvas Authoring Source Basis

== Краткое описание
Карточка фиксирует source basis child domain `canvas-authoring`: canvas handlers, properties panel rendering, object editing logic и interaction behavior внутри web bundle v2.4.

== Текущее состояние
Локальный source basis выделен из общего `SRC-VSS-EDITOR-001`, чтобы child domain имел собственную опору в слое `24` и не терял автономность при навигации и review.

== Что подтверждено
- `app.js` содержит draw/render pipeline, properties panels и handlers для drag, selection, zoom и pan.
- Основная логика child domain сосредоточена в `app.js`, а входные элементы UI - в `index.html`.
- Отдельных внешних источников или backend contracts для этого child domain не найдено.
