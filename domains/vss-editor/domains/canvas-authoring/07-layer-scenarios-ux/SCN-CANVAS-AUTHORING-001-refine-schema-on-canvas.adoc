= Refine Schema On Canvas

== Краткое описание
В этом дочернем домене Актор последовательно уточняет рабочую схему прямо на canvas: двигает блоки, меняет свойства, добавляет объекты, проверяет композицию и сразу видит результат каждого шага в одном интерактивном рабочем поле.

== Текущее состояние
Сценарий уже реализован как непрерывное взаимодействие между canvas и правой панелью свойств. Работа не дробится на отдельные экраны, а держится как один authoring loop вокруг текущей схемы.

== Что подтверждено
- Canvas поддерживает drag, selection, rectangle selection, zoom и pan.
- Properties panel поддерживает schema, node, flow и text editing.
- Локальные graph objects и их связи меняются прямо внутри открытой рабочей схемы.

== Замечания аналитика
Детализированный parent-level route уже materialized как `SCN-VSS-EDITOR-004`. Эта child-domain карточка удерживает ту же предметную ветку на уровне decomposition и Antora navigation.
