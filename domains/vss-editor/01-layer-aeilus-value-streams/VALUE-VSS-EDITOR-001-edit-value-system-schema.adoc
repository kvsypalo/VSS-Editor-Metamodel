= Edit Value System Schema

== Краткое описание
Ценность этого продукта в том, что актер может быстро собрать понятную рабочую картину системы: где создается ценность, где возникают потери, как связаны converters и какие элементы реально проходят по flows. В результате у команды появляется единая схема, через которую удобно обсуждать устройство системы и уточнять ее без повторного ручного восстановления.

Для актора редактор ценен тем, что позволяет в одной сессии собрать схему, уточнить ее вместе с коллегами, вернуться к уже сделанной работе через JSON и выпустить результат в виде переносимого артефакта. Продукт дает редактируемую и воспроизводимую схему value system, с которой можно продолжать работу и обсуждение.

== Текущее состояние
В v2.4 поток ценности реализован как локальная работа в одном браузерном окне без серверной части. Актер может начать новую схему, продолжить уже подготовленную, объединить несколько фрагментов в одном canvas и довести результат до состояния, пригодного для повторного открытия или обсуждения вне редактора.

== Что подтверждено
- Help modal прямо описывает schema как модель движения value через converters и flows.
- Toolbar и properties panel позволяют без выхода из редактора уточнять состав схемы, связи и параметры объектов.
- Export JSON и Export PNG превращают текущую схему в переносимый рабочий результат, который можно открыть снова или показать другим участникам обсуждения.

== Замечания аналитика
Source basis не формализует роль актора, но по предмету продукта actor читается как аналитик, архитектор или другой участник, который описывает систему через схему движения ценности.

== Требуемые изменения
Новых product changes не запрошено. Требуется зафиксировать value movement текущего продукта.
