= LAYER-CANONICAL-TABLES-002

Этот документ задает только правило существования канонической таблицы слоев и не хранит саму таблицу.

Детальная semantics, object prefixes, special fields, traceability rules и authoring notes каждого слоя хранятся непосредственно в соответствующем файле внутри `00-meta/layers/`.

Каноническая таблица полного набора слоев хранится в `../layers-summary/LAYER-CANONICAL-TABLES-002.adoc`.

Примечания:
- этот документ не хранит layer semantics и не дублирует summary tables
- слой `00-meta` не является предметным слоем домена и используется для локального meta/context уровня
- новые канонические слои добавляются только в хвост числового списка без перенумерации уже принятой части канона
- при добавлении, удалении или изменении слоя должны быть обновлены и `../layers/`, и `../layers-summary/`
