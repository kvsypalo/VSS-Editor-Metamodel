= Workspace Lifecycle Trace

== Краткое описание
Карточка фиксирует локальную цепочку child domain `workspace-lifecycle`: родительский process приводит Актора к рабочей схеме, child-domain scenario удерживает file/session lifecycle, локальный requirement формализует contract, а детальные parent-level scenarios и tests дают подтверждающее раскрытие веток `New`, `Open`, `Add Schema`, `Export` и invalid-JSON refusal.

== Текущее состояние
Цепочка child domain уже читается из существующего product scope, но теперь получила собственную `TRACE-*` карточку для автономной review и Antora navigation в слое `25`.

== Что подтверждено
- Child domain держится на одном локальном scenario и одном локальном requirement.
- Детализация локальной цепочки раскрывается через parent-level `SCN-VSS-EDITOR-001/002/003/005/006`.
- Failure branch замыкается существующим validating test `TEST-VSS-EDITOR-006`.

== Замечания аналитика
Карточка намеренно использует связи вверх к родительскому домену, потому что decomposition не создает новый product route, а выделяет локальный участок общего маршрута.

== Требуемые изменения
Новых изменений не требуется.
