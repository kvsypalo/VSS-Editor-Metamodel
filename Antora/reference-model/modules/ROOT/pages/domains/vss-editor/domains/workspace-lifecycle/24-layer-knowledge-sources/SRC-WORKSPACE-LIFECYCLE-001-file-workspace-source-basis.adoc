= File Workspace Lifecycle Source Basis

== Краткое описание
Карточка фиксирует source basis child domain `workspace-lifecycle`: обработчики schema menu, file input, import/export branches и отказную ветку по невалидному JSON внутри web bundle v2.4.

== Текущее состояние
Локальный source basis выделен из общего `SRC-VSS-EDITOR-001`, чтобы child domain имел собственную опору в слое `24` и не терял автономность при навигации и review.

== Что подтверждено
- `btnNew`, `btnOpen`, `btnImport`, `btnSaveFile`, `btnExportPng` и `fileInput.onchange` покрывают локальный scope домена.
- Основная логика child domain сосредоточена в `app.js`.
- Отдельних внешних источников или backend contracts для этого child domain не найдено.

== Замечания аналитика
Карточка не создает новый внешний источник, а локально выделяет подтвержденный срез общего source basis родительского домена.

== Требуемые изменения
Новых изменений не требуется.
