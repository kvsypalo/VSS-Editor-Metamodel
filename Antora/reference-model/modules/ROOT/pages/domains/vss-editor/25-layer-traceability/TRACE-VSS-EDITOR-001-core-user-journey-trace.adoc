= Core User Journey Trace

== Краткое описание
Карточка фиксирует основную сквозную цепочку домена: Актор получает прикладную ценность через работу в browser application, проходит процесс от появления редактируемой схемы до ее экспорта, а маршрут раскрывается через человекочитаемые сценарии, требования и validating tests. Дополнительно здесь зафиксированы access- и critical-path assessments, чтобы ключевой пользовательский маршрут не оставался без явной оценки по слоям `31` и `33`.

== Текущее состояние
Цепочка первой wave опирается на один верхнеуровневый value stream, один основной process object, одну application card, шесть user scenarios, четыре requirements и шесть validating tests. Для same-wave completeness добавлены baseline access assessment и critical-path assessment ключевого маршрута.

== Что подтверждено
- `VALUE-VSS-EDITOR-001` связан с `GOAL-VSS-EDITOR-001` и `PROC-VSS-EDITOR-001`.
- Каждый materialized `SCN-*` имеет хотя бы один явный `TEST-*`.
- Code-driven scope дополнительно разложен на `APP-*`, `RULE-*`, `DATA-*`, `CODE-*`, `ACCESS-*` и `CRIT-*`.

== Замечания аналитика
Карточка фиксирует только core chain первой wave и не претендует на полное покрытие всех возможных secondary links между объектами.

== Требуемые изменения
Новых изменений не требуется.
