= Local Single-User Access Baseline

== Краткое описание
Слой доступа фиксирует простую базовую модель продукта: редактор рассчитан на локальную однопользовательскую сессию, где все команды доступны тому, кто открыл страницу в браузере. Права на чтение и выбор файлов делегированы браузеру и локальной операционной системе.

== Текущее состояние
Внутри интерфейса действует единый набор команд для текущей сессии: `New`, `Open`, `Add Schema`, `Export JSON` и `Export PNG` доступны одному локальному пользователю, работающему со своей схемой в браузере.

== Что подтверждено
- Все ключевые команды редактора доступны из одного toolbar и одной схемной сессии.
- Работа со схемой строится вокруг одной локальной пользовательской сессии.
- Файловые операции выполняются через browser file picker и скачивание файлов под контролем браузера и локальной ОС.

== Замечания аналитика
Карточка фиксирует текущую access-модель как самостоятельное baseline-свойство продукта: локальная однопользовательская сессия с единым набором команд внутри редактора.

== Требуемые изменения
Новых изменений не требуется.
