= Static Browser Editor Bundle

== Краткое описание
Реализация продукта сосредоточена в статическом browser bundle без build-step: `index.html` задает shell интерфейса, `app.js` содержит почти всю предметную и UI-логику редактора, `styles.css` оформляет editor layout. Bundle работает как single-page application поверх SVG canvas и in-memory schema state.

== Текущее состояние
Кодовая база v2.4 компактна и почти полностью сосредоточена в одном JavaScript-файле. Внутри него объединены model creation, normalization, render pipeline, properties panel, import/export, undo и interaction handlers для canvas.

== Что подтверждено
- `index.html` объявляет toolbar, schema menu, canvas, properties panel, help modal и status bar.
- `app.js` реализует весь editor runtime: state, drawing, selection, drag, import/export, help modal, auto-load schema from query.
- `styles.css` фиксирует editor-like layout с отдельными областями toolbar, canvas, properties panel и status bar.
