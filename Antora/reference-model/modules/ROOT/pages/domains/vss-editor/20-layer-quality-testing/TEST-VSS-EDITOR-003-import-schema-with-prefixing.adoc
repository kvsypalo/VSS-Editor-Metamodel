= Import Schema With Prefixing Test

== Краткое описание
Тест проверяет, что `Add Schema` добавляет внешний JSON в текущий canvas и корректно remap-ит colliding identifiers. Проверка закрывает import scenario с merge semantics текущей реализации.

== Текущее состояние
Тест materialized как validating specification на функцию `importWithPrefix()` и связанный user route через `Schema` menu.

== Что подтверждено
- Import использует отдельный режим `fileMode = "import"`.
- При коллизиях node, flow, element и text IDs получают префикс.
- Связи между imported objects переназначаются на remapped IDs.

== Замечания аналитика
Тест не утверждает бизнесовые merge rules beyond ID remapping и append behavior, потому что другой semantics в коде не найден.

== Требуемые изменения
Новых изменений не требуется.
