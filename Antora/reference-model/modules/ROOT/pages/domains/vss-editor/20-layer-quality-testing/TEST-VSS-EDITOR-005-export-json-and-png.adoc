= Export JSON And PNG Test

== Краткое описание
Тест проверяет, что editor умеет выгружать текущую схему как JSON и как PNG-изображение диаграммы. Проверка закрывает export scenario с двумя portable output variants.

== Текущее состояние
Тест materialized как validating specification по `btnSaveFile` и `exportPng()`.

== Что подтверждено
- JSON export сериализует весь `state.schema`.
- PNG export формирует SVG из `contentGroup`, рендерит его в canvas и скачивает результат как PNG.
- При отсутствии diagram content PNG export отказывается с сообщением `Nothing to export`.

== Замечания аналитика
Тест охватывает обе export-команды в одной проверке, потому что они разделяют один user route и один набор acceptance criteria на переносимость результата.

== Требуемые изменения
Новых изменений не требуется.
