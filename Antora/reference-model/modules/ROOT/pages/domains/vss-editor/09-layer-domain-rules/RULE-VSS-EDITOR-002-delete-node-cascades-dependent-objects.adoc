= Delete Node Cascades Dependent Objects

== Краткое описание
Удаление converter node автоматически удаляет все flows, которые ссылаются на него как на source или target, и все element types, размещенные внутри этого node. Правило сохраняет целостность graph state после удаления узла.

== Текущее состояние
Правило реализовано в `deleteNode(nodeId, push = true)`. Функция фильтрует `state.schema.flows`, `state.schema.elements` и `state.schema.nodes`, а затем очищает selection и рендерит schema-level view.

== Что подтверждено
- Flows удаляются, если их `source_node_id` или `target_node_id` совпадает с удаляемым node.
- Elements удаляются, если `location === "in_node"` и `node_id` совпадает с удаляемым node.
- После каскадного удаления editor возвращается к selected schema state.

== Замечания аналитика
Source basis не показывает soft-delete или восстановление удаленного узла beyond global undo history.

== Требуемые изменения
Новых изменений не требуется.
