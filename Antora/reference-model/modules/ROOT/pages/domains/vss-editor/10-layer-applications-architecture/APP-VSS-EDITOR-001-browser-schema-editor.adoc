= VSS Editor Browser Application

== Краткое описание
`VSS Editor` как приложение - это одностраничный браузерный редактор, в котором Актор открывает, собирает, уточняет и выгружает схему value system. Для домена это не абстрактная техническая оболочка, а конкретная прикладная среда, где сходятся toolbar, canvas, свойства объектов, import/export и help-модель продукта.

== Текущее состояние
В текущей версии приложение работает локально в браузере и не опирается на отдельный backend или build pipeline. Вся основная предметная работа проходит внутри одного UI shell с canvas и правой панелью свойств.

== Что подтверждено
- `index.html` задает toolbar, schema menu, canvas, properties panel, status bar и help modal.
- `app.js` реализует состояние схемы, рендер диаграммы, команды меню и взаимодействие с canvas.
- `styles.css` оформляет editor-like layout с четким разделением рабочей области и панели свойств.

== Замечания аналитика
Продукт small-scale, поэтому в этой wave приложение materialized одной `APP-*` карточкой без дальнейшего разбиения на внутренние subsystem cards.
