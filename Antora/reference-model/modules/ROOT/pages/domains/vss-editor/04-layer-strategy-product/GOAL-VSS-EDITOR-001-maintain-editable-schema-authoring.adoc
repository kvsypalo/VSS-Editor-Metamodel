= Maintain Editable Schema Authoring

== Краткое описание
Продуктовая цель редактора состоит в том, чтобы Актор мог быстро собрать и поддерживать редактируемую схему value system внутри одного рабочего контура. Цель выражается в непрерывном authoring loop между canvas, properties panel и командами импорта-экспорта.

== Текущее состояние
VSS Editor v2.4 уже предоставляет минимальный authoring loop внутри одной страницы: schema menu управляет жизненным циклом файла, canvas отражает структуру схемы, properties panel позволяет менять свойства выбранного объекта.

== Что подтверждено
- В UI присутствуют точки входа для создания, открытия, импорта и экспорта схемы.
- Состояние выбранного schema object напрямую отображается в properties panel.
- Undo и режимы `Hand` / `Arrow` поддерживают короткий цикл уточнения схемы.

== Замечания аналитика
Цель выводится из продукта и UI. В текущей wave она фиксируется как code-derived product goal.

== Требуемые изменения
Новых изменений не требуется.
