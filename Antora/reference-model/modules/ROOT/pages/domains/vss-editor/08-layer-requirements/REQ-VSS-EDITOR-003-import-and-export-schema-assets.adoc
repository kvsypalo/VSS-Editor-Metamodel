= Import And Export Schema Assets

== Краткое описание
Редактор должен позволять объединять внешнюю JSON-схему с текущим canvas и выгружать результат работы в переносимые артефакты. Поддерживаются import существующей схемы через `Add Schema` и export результата в JSON или PNG.

== Текущее состояние
`Add Schema` запускает режим `import`, читает JSON-файл и через `importWithPrefix()` добавляет imported objects в текущую схему. `Export JSON` сериализует `state.schema`, а `Export PNG` строит отдельный SVG, рендерит его на canvas и скачивает PNG с подписью `Æilus`.

== Что подтверждено
- Import добавляет nodes, flows, elements и texts к текущему состоянию вместо полной замены схемы.
- Export JSON формирует файл с именем по `state.schema.schema.title`.
- Export PNG рендерит только diagram content и добавляет branded footer.

== Замечания аналитика
Requirement фиксирует только подтвержденные import/export форматы текущей реализации. Дополнительные interchange formats в source basis не обнаружены.

== Требуемые изменения
Новых изменений не требуется.
