= Edit Schema On Canvas

== Краткое описание
Это центральный пользовательский маршрут редактора. Актор смотрит на схему как на рабочую поверхность, двигает блоки, переименовывает их, меняет связи и параметры потоков, добавляет пояснения и постепенно доводит схему до нужного состояния, не покидая canvas.

== Текущее состояние
Сценарий уже реализован как непрерывная работа на canvas и в правой панели свойств. Актор может править схему короткими шагами: выделить объект, изменить параметры, подвигать блоки, проверить композицию целиком, добавить новый элемент и сразу увидеть результат на рабочей поверхности.

== Что подтверждено
- Редактор поддерживает отдельные формы для schema, node, flow и text.
- Актор может менять имена, описания, связи, числовые параметры, цвета и другие свойства объектов.
- Canvas поддерживает drag, выделение рамкой, панорамирование и масштабирование, поэтому схема редактируется как единое рабочее поле.

== Замечания аналитика
Внутри маршрута есть несколько близких микродействий, но они образуют один человекочитаемый сценарий: Актор уточняет текущую схему до нужного состояния и сразу видит итог каждого изменения.

== Требуемые изменения
Новых изменений не требуется.
