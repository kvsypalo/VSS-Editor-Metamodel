= Open Schema From JSON

== Краткое описание
Актор возвращается к уже подготовленной схеме: выбирает `Open`, подает JSON-файл и получает знакомую рабочую картину обратно на canvas. Сценарий нужен для продолжения работы без пересборки схемы с нуля.

== Текущее состояние
Сценарий уже реализован и заменяет текущую рабочую схему содержимым выбранного файла. После загрузки редактор показывает открытую схему как новую основу для дальнейшего редактирования.

== Что подтверждено
- `btnOpen` переводит file input в режим `open`.
- При успешном чтении JSON редактор полностью заменяет текущую схему содержимым файла.
- После загрузки схема заново показывается в центре рабочей области.

== Замечания аналитика
Отдельная предметная валидация структуры схемы beyond JSON parse и normalization в source basis не подтверждена.

== Требуемые изменения
Новых изменений не требуется.
