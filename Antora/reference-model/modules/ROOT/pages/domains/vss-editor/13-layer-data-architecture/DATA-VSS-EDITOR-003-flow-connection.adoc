= Flow Connection

== Краткое описание
Flow connection описывает направленную связь между двумя converters или между converter и внешним миром. Объект хранит endpoints, количественную value-оценку, unit, цвет, label visibility и список routed element types.

== Текущее состояние
Flow хранится в массиве `flows`, редактируется через flow properties panel и визуализируется как curve или polyline с arrowhead. Тип flow пересчитывается автоматически по наличию source/target node IDs.

== Что подтверждено
- Flow содержит `id`, `name`, `type`, `source_node_id`, `target_node_id`, `value`, `unit`, `color`, `show_label`, `element_type_ids`, `attributes`.
- Flow может быть `external_in`, `external_out` или `internal`.
- Flow routing и visual geometry зависят от endpoint configuration и относительного положения nodes.

== Замечания аналитика
Source basis не показывает отдельную persistence-валидацию flow constraints вне runtime logic редактора.

== Требуемые изменения
Новых изменений не требуется.
