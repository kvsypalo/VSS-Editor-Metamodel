= Converter Node

== Краткое описание
Converter node представляет функциональный блок схемы value system. Он имеет идентификатор, имя, позицию и размер на canvas, описание и дополнительные ссылки, а также выступает контейнером для in-node element types.

== Текущее состояние
Node присутствует как отдельный объект в массиве `nodes` и редактируется через node properties panel. Новый node может быть создан как следующий converter относительно выбранного узла.

== Что подтверждено
- Node содержит `id`, `name`, `position`, `size`, `description`, `bpmn_link`, `link`, `attributes`.
- Node визуально рендерится как прямоугольник с wrapped text label.
- Node может быть источником и приемником flows и контейнером in-node elements.

== Замечания аналитика
Help modal описывает converter как functional block, но не выделяет более глубокую типизацию converter roles.

== Требуемые изменения
Новых изменений не требуется.
