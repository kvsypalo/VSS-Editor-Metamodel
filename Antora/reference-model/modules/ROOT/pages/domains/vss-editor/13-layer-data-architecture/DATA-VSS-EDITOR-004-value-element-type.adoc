= Value Element Type

== Краткое описание
Value element type описывает, что именно движется через converters и flows в схеме. Объект хранит имя элемента, признак value или anti-value, знак, quantity, unit, location и привязку либо к node, либо к flow.

== Текущее состояние
Element types живут в массиве `elements`, редактируются внутри node properties panel и участвуют в flow routing selection. В default schema уже присутствует `Customer Value` как базовый пример такого объекта.

== Что подтверждено
- Element содержит `id`, `name`, `kind`, `sign`, `quantity`, `unit`, `location`, `node_id`, `flow_id`, `attributes`.
- UI различает `Value` и `Anti-value` и пересчитывает `sign` в зависимости от выбора.
- Flow routing selection оперирует именно IDs element types связанного converter.

== Замечания аналитика
Source basis использует термин `element types`, хотя фактически объект хранит не только тип, но и quantity instance parameters текущей схемы.

== Требуемые изменения
Новых изменений не требуется.
