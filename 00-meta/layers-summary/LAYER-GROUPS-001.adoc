= LAYER-GROUPS-001

Обзорные группы канонического набора слоев.

Этот документ хранит grouped view по всему стеку и должен обновляться вместе с `../layers/` и другими summary docs при изменении канона.

== Layer groups
- `00` — meta/context уровень домена
- `01` — поток ценности
- `02-04` — контекст, стейкхолдеры и стратегия
- `05-09` — бизнес-структура, процессы, сценарии, требования и правила
- `10-15` — архитектура, интерфейсы, интеграции, данные, код и технологии
- `16-21` — эксплуатация, качество, риски, тесты и наблюдаемость
- `22-25` — изменения, задачи, знания и traceability
- `26-33` — гипотезы, валидация и continuity-aware assessment

== Назначение группировки
- помогать навигации по канону
- помогать структурному обзору слоев как набора
- не заменять детальные per-layer rules из `../layers/`
