= LAYER-SPECIAL-FIELDS-001

Агрегированная summary layer-specific fields по всему каноническому стеку.

Базовая схема карточки задается `../schemas/BASE-CARD-SCHEMA-001.adoc`. Этот файл хранит только то, что добавляется поверх базовой схемы на уровне конкретных слоев.

При изменении layer-specific fields или допустимых значений этот summary должен обновляться вместе с `../layers/`.

== Shared model для слоев `27-33`
- `record_mode`: `baseline_constraint`, `initiative_assessment`, `mixed`
- `hard_constraints_text`: постоянные must-have ограничения слоя
- `context_text`: локальный assessment context, допущения и применимость
- `assessment_subject_refs`: ссылки на assessment subject
- `affected_requirement_refs`: ссылки на затронутые requirement objects

== Shared decision values для слоев `27-33`
- `within_constraints`
- `changes_required`
- `constraint_violation`
- `unknown`
- `not_applicable`

== Critical-path-aware enums
- `critical_path`: `true`, `false`
- `criticality_level`: `mission-critical`, `business-critical`, `important`, `supporting`
- `scenario_type`: `primary`, `alternate`, `failure`, `degraded`, `recovery`
- `criticality`: `critical-path`, `high`, `medium`, `low`
- `critical_path_metric`: `true`, `false`
