= 24 Knowledge Sources

== Path
`domains/<domain>/24-layer-knowledge-sources/`

== Purpose
Слой хранит источники знаний, evidences, основания решений и карточки, описывающие артефакты или внешние источники истины.

== Activation
- Этот слой входит в канонический стек `01-33`.
- Каждый дочерний домен должен иметь собственные объекты в этом слое.

== Typical objects
- `SRC-*`
- `ART-*`
- `REF-*` при необходимости локальной навигации

== Special fields
- Отдельные layer-specific поля не заданы. Используется `BASE-CARD-SCHEMA-001`.

== Traceability rules
- Каждый `PROC-*` должен иметь хотя бы один `SRC-*`.
- Каждый дочерний домен должен иметь собственные объекты минимум в `00-meta`, `08`, `24` и `25`.

== Key rules
- В этом слое оформляются источники, на которые реально опирается анализ.
- Файлы артефактов живут в `artifacts/`, а карточки о них и ссылки на них живут здесь.

== Related layers
- Upstream: практически все слои модели, если они опираются на источники.
- Downstream: `25 Traceability`.

== Authoring notes
- Физические файлы артефактов живут в `artifacts/`, а карточки артефактов и ссылки на них живут в этом слое.
- `REF-*` создается только если прямой связи на объект выше по иерархии недостаточно для локальной навигации.
